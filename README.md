# Sobre o Projeto

Este repositório contém o código-fonte do meu site e portfólio profissional, desenvolvido com foco em desempenho, posicionamento digital e conversão estratégica.

O projeto foi concebido para ir além de uma apresentação visual convencional. A proposta central é construir um ecossistema digital de alta performance, unindo desenvolvimento web moderno, experiência do usuário (UI/UX) e estratégias de tráfego pago aplicadas ao ambiente comercial.

Toda a estrutura foi planejada para fortalecer autoridade profissional, otimizar a experiência de navegação e direcionar usuários qualificados para canais de conversão direta.

---

# Objetivos da Plataforma

A aplicação possui como principais objetivos:

* fortalecer presença digital e posicionamento profissional;
* apresentar portfólio, habilidades e serviços de forma estratégica;
* demonstrar capacidade técnica em desenvolvimento Front-End;
* integrar design, performance e marketing digital em um único ambiente;
* maximizar geração de leads através de fluxos otimizados de conversão.

---

# Tecnologias Utilizadas

O projeto foi desenvolvido utilizando tecnologias Front-End nativas, priorizando código limpo, desempenho e carregamento otimizado.

## HTML5

Responsável pela estruturação semântica da aplicação, garantindo organização adequada do conteúdo, acessibilidade e melhor rastreabilidade para mecanismos de busca.

---

## CSS3

Utilizado para estilização moderna e responsiva da interface, incluindo:

* variáveis globais (`:root`);
* CSS Grid Layout;
* Flexbox;
* animações nativas;
* Media Queries;
* transições visuais e efeitos de interação.

A identidade visual foi construída visando sofisticação, clareza e alta legibilidade em diferentes dispositivos.

---

## JavaScript Vanilla

Toda a lógica de interatividade foi implementada em JavaScript puro, sem dependência de frameworks externos.

Entre os recursos desenvolvidos, destacam-se:

* menu responsivo para dispositivos móveis;
* manipulação dinâmica do DOM;
* animações ativadas por rolagem;
* contadores dinâmicos;
* utilização de `IntersectionObserver` para otimização de desempenho visual.

# Diferenciais do Projeto

## Alta Performance

A estrutura do projeto foi otimizada para garantir carregamento rápido e excelente desempenho.

Entre as práticas adotadas:

* separação organizada de arquivos;
* otimização de imagens;
* redução de processamento desnecessário;
* utilização de código leve e nativo.

---

## Design Responsivo

A aplicação adapta automaticamente sua interface para:

* smartphones;
* tablets;
* notebooks;
* monitores de alta resolução.

O objetivo é garantir consistência visual e experiência fluida em qualquer ambiente de navegação.

---

## Interface Estratégica para Conversão

A estrutura visual da plataforma foi desenvolvida com foco em princípios de UI/UX voltados para marketing e vendas.

Foram aplicados conceitos como:

* escaneabilidade visual;
* hierarquia de informações;
* posicionamento estratégico de CTAs;
* psicologia de cores;
* redução de atrito na navegação.

---

## Integração Direta com WhatsApp

A plataforma possui integração direcionada para conversão imediata via WhatsApp, facilitando o contato rápido com potenciais clientes e reduzindo etapas no processo comercial.
